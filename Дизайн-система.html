Лаймовая система — две темы

Визуальный язык приложения: тёплый светлый фон или глубокий тёмный, единый лаймовый акцент (#C6F24E), скруглённые карточки и контрастные «акцентные» карточки. Все компоненты построены на CSS‑переменных — переключение темы вверху справа меняет одну переменную‑тему, не трогая разметку. Ниже — токены, компоненты с состояниями и заметки по доступности.

01

Токены цвета

Семантические токены, а не «голые» хексы. Слева указано имя переменной, ниже — значения в светлой (L) и тёмной (D) теме. Свотчи живые — меняются вместе с темой.

Лайм — единственный акцент: используется для главных действий, активных состояний и данных. Не раскрашивать им всё подряд — сила в сдержанности.

08

Бейджи сетей

Единый размер и радиус, бренд‑цвет сети. Тёмные бейджи (TikTok / Threads / X) в тёмной теме автоматически осветляются, чтобы оставаться читаемыми. В продакшене иконки заменяются официальными.

09

Карточки

Базовая карточка — белая/тёмная поверхность, мягкая тень, крупный радиус. Feature‑карточка — контрастная «акцентная» поверхность для главных метрик и статусов.

Базовая карточка
Заголовок секции

Контент, списки, графики. Используется для большинства блоков.

Feature‑карточка
Акцентный блок

Главная метрика, тренды, статус системы. Не более 1–2 на экран.

14

Доступность и принципы

Доступность

• Видимый фокус (лаймовая обводка) на всех интерактивных элементах.

• Контраст текста ≥ 4.5:1 в обеих темах; статус не передаётся только цветом — есть текст/иконка.

• Клавиатура: Tab/Shift+Tab, Enter/Space для действий, Esc для закрытия.

• prefers-reduced-motion отключает анимации.

Принципы

• Семантические токены, без «голых» хексов в компонентах.

• Лайм — один акцент: главные действия, активное, данные.

• Лейбл = действие; единый словарь по всему продукту.

• Пустое состояние — приглашение к действию, ошибка — что случилось и как исправить.

15

Токены для разработчика (CSS)

Готовый блок переменных. Тема переключается атрибутом data-theme на корне — разметка и компоненты не меняются.

/* светлая тема (по умолчанию) */
:root{
  --bg:#ECECE6; --surface:#FFFFFF; --surface-2:#F4F4EE;
  --feature:#17181A; --feature-ink:#FFFFFF;
  --ink:#17181A; --muted:#71757C; --on-accent:#16180D;
  --line:#E8E8E1;
  --accent:#C6F24E; --accent-deep:#B6E63B;
  --green:#19A45B; --red:#E0533D; --warn:#C9892A;
  --shadow:0 1px 2px rgba(20,20,20,.05), 0 10px 30px rgba(20,20,20,.05);
  --r:22px; --r-sm:14px; --r-xs:10px;
  --sp-1:4px; --sp-2:8px; --sp-3:12px; --sp-4:16px; --sp-5:18px; --sp-6:24px;
}
/* тёмная лаймовая тема */
[data-theme="dark"]{
  --bg:#101208; --surface:#1A1D14; --surface-2:#23271B;
  --feature:#0B0D07; --feature-ink:#F3F5EC;
  --ink:#EEF1E4; --muted:#9CA291; --on-accent:#16180D;
  --line:#2C3122;
  --accent:#C6F24E; --accent-deep:#A9DB36;
  --green:#37C470; --red:#F0795F; --warn:#E0A93E;
  --shadow:0 1px 2px rgba(0,0,0,.5), 0 14px 34px rgba(0,0,0,.4);
}
