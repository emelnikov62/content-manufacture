# Контент‑Завод — пакет для команды

Веб‑приложение для управления соцсетями нескольких бизнес‑направлений: тренды → AI‑генерация
(текст/фото/видео) → планирование → автопостинг → аналитика, плюс воронки авто‑ответа.

## С чего начать
Откройте **index.html** — это стартовая страница со ссылками на все материалы.

## Состав пакета
| Файл | Что это | Для кого |
|------|---------|----------|
| index.html | Стартовая страница пакета | Все |
| ТЗ.html / ТЗ.md | Техническое задание (концепция, API, фазы, бюджет) | Разработка, PM |
| Дизайн-референс_экраны.html / .md | Постраничное описание всех экранов | Дизайн, PM |
| Дизайн-система.html | Токены и компоненты, 2 темы (светлая/тёмная) | Дизайн, фронтенд |
| Прототип_все_экраны.html | Интерактивный прототип всех экранов меню | Все |
| Прототип_дашборд.html | Отдельный прототип дашборда | Все |

HTML‑файлы открываются в любом браузере (двойной клик), без интернета.
.md — исходники документов для правок (GitHub/IDE).

## Ключевые решения
- Сети: Instagram, TikTok, Telegram, Threads, Facebook, X.
- Публикация и личка: **Postproxy** (единый провайдер, аудит TikTok пройден).
- AI‑генерация: **kie.ai** (текст, изображения, видео, аудио — один API).
- Тренды: **EnsembleData**.
- Бюджет внешних сервисов: **$100–300/мес** с учётом расходов и лимитами в приложении.
- Фазы: 1) ядро/MVP · 2) контент‑завод + воронки (вкл. видео) · 3) опционально.

_Иконки соцсетей в прототипе — упрощённые оригинальные марки; в продакшене заменяются официальными._
